# Video Listing

## 1. About
React + Vite app for a video listing UI.

## 2. Local Setup
```bash
npm install
npm run dev
```

## 3. Production Build
```bash
npm run build
```

## 4. Vercel Deploy
- Root Directory: `Projects/video-listing` (or `.` if this folder is repo root)
- Build Command: `npm run build`
- Output Directory: `dist`
